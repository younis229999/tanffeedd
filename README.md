# معالج ملفات التوطين — Salary Localization Processor

تطبيق سطح مكتب (Windows) لمعالجة ملفات Excel الخاصة بتوطين الرواتب:
**تنظيف البيانات + التحقق والتنبيهات + دمج المكررات + توليد ملف نهائي + تقرير PDF احترافي**.
الواجهة والتقرير عربيان (RTL)، وملف الإكسل الناتج إنجليزي (LTR).

---

## المزايا

- **تنظيف رقم الإضبارة** إلى الصيغة `YYYY_NN` (مثل `2025_65`) مع تطبيع الأرقام العربية إلى غربية.
- **منع تكرار رقم الإضبارة** تلقائياً (أصغر رقم متاح لنفس السنة) مع عرض جدول التغييرات (القديم ← الجديد).
- **دمج المكررات حسب الآيبان فقط** وجمع المبالغ — لا يُدمج تطابق الأسماء، ويُنبَّه على تعارض الاسم لنفس الآيبان.
- **تنبيهات ملوّنة**: طول الآيبان، النمط العراقي، الآيبان الملغي، المبلغ المرتفع، الحقول الناقصة، وفحص MOD-97 الاختياري.
- **مخرجات**: ملف Excel نهائي (LTR) + تقرير PDF عربي بالتذييل المطلوب.
- **إعدادات قابلة للتعديل**: اسم المديرية، قائمة الآيبانات الملغية، العتبات، ربط الأعمدة.
- لا يُكتب فوق الملف الأصلي إطلاقاً + نسخ احتياطية لقائمة الآيبانات + سجل عمليات (logs/).

---

## التشغيل من المصدر

```bash
pip install -r requirements.txt
python main.py
```

> يتطلب خطاً عربياً في `assets/fonts/Amiri.ttf` (مضمّن).

---

## التغليف إلى ملف exe (على ويندوز)

```bash
pip install -r requirements.txt
pyinstaller build.spec
```

الناتج: `dist/SalaryProcessor.exe` — يعمل دون تنصيب Python.
يُحفظ `config/settings.json` بجوار الملف التنفيذي (إعدادات دائمة).

## بنية المشروع

```
salary_processor/
├── main.py                 # نقطة التشغيل (RTL + خط عربي)
├── ui/
│   ├── main_window.py      # اختيار ملف ← معاينة ← معالجة ← نتائج
│   ├── results_view.py     # تبويبات النتائج + التصدير
│   └── settings_dialog.py  # الإعدادات
├── core/
│   ├── settings.py         # إدارة settings.json + نسخ احتياطية
│   ├── excel_loader.py     # قراءة/كتابة Excel (الناتج LTR)
│   ├── folder_number.py    # تنظيف وتفريد أرقام الأضابير
│   ├── validator.py        # التحقق والتنبيهات
│   ├── merger.py           # دمج المكررات حسب الآيبان
│   ├── processor.py        # المنسّق (خط المعالجة)
│   ├── report.py           # تقرير PDF عربي
│   └── logger.py           # سجل العمليات
├── config/settings.json
├── assets/fonts/Amiri.ttf  # خط عربي للـ PDF (OFL)
├── tests/                  # اختبارات المنطق والواجهة
├── requirements.txt
└── build.spec              # إعداد PyInstaller
```

## ملاحظات

- خريطة الأعمدة الافتراضية: A=رقم الإضبارة، E=المبلغ، H=الاسم، I=الآيبان (قابلة للتغيير من الإعدادات).
- الآيبان العراقي: يبدأ بـ `IQ` بطول 23 خانة (قابل للتعديل).
- خط Amiri مرخّص برخصة OFL (حر الاستخدام والتوزيع).
